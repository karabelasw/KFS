  <h4>The Karabelas File System (KFS) is being constructed as a web based software application and a long-term software development exercise. It serves as an opportunity to explore modern (not that modern) tools to assist in database design, object-oriented programming, and knowledge management while simultaneously evaluating the effectiveness of artificial intelligence as assistant in the design, implementation, documentation, and evolution of a complex software system.
KFS is both a web application product and an experiment. Its success will be measured by the usefulness of the application, as well as the knowledge gained throughout its design and development.
